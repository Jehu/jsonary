What is Jsonary?

Jsonary is a library for quickly building JSON-based AJAX clients, particularly for APIs that are described by JSON Schema.

Schemas not only places constraints on the data, but can also define hyperlinks. An API correctly described with JSON Schema can be navigated using these hyperlinks. Jsonary uses the hyperlinks (along with any meta-data in the schema) to provide a "universal client" framework, allowing customisation for extra requirements such as OAuth tokens.

Each JSON Schema also has a URI, so it is possible to add renderers for particular parts of the API, so that the "universal client" can be easily adapted to provide a custom user-interface.

What is JSON Schema?

JSON Schema is a format for describing JSON data. It can restrict the form that the data can take (for instance, requiring/banning properties, minimum/maximum numeric values, etc.) as well as providing useful extra information, such as hyperlinks, or documentation. The main site is json-schema.org.

Schemas can reference/link to other schemas, and specify complex behaviours - for example, conditional schemas that only apply if a particular property is present, or lists of sub-schemas of which only one may apply.

If you want to understand more about JSON Schema, this page provides documentation for the format, along with interactive examples for various keywords.
